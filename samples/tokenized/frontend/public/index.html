Tokenized Charge - Information

This is a Demo to demonstrate Tokenized Charges. Tokenized Charges can be carried out on the Card API (i.e) you can only tokenize cards and not account numbers.
Go here to read more on how to charge a saved card.

  • In this demo, the amount being charged is NGN5000.
  • You perform the card charge first.
  • Once the card charge is successful, copy the card token returned(Please note that in a real world scenario, you're meant to securely save the card token for subsequent charging) and paste in the the card token field.
  • Charge the Card Token
  • Review the generated codebase

Please Note That: The demo card below is a Mastercard and hence this demo does not demonstrate how to handle VISA cards. Kindly see the CLI documentation on how to handle VISA cards
